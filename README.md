# Online Bookstore SQL Analysis Project

## Project Overview

This project analyzes an Online Bookstore database using SQL.

The project demonstrates:

- Database Creation
- Table Relationships
- Data Import
- SQL Queries
- Data Analysis
- Business Insights
- Window Functions
- Subqueries
- Aggregations
- KPI Reporting


## Database Schema

Tables Used:

1. Books
2. Customers
3. Orders

### Relationships

Orders → Customers (Customer_ID)

Orders → Books (Book_ID)
